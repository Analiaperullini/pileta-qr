# 🏊‍♂️ Sistema de Gestión de Socios y Acceso QR - Pileta Municipal

Sistema integral desarrollado en **Spring Boot 3** para la gestión de socios, administración de deportes y validación de acceso mediante carnets digitales con códigos QR.

## 🚀 Funcionalidades Principales

* **Registro de Socios:** Alta de socios con carga de fotos y datos personales.
* **Gestión de Deportes:** Vinculación dinámica de socios con múltiples disciplinas.
* **Carnet Digital:** Generación automática de carnet en formato HTML (Thymeleaf) listo para imprimir.
* **Acceso mediante QR:** Sistema de tokens únicos mensuales para validar el ingreso en portería.
* **Buscador Avanzado:** Filtros por nombre, DNI y deporte.

## 🛠️ Tecnologías Utilizadas

* **Backend:** Java 17, Spring Boot 3, Spring JDBC.
* **Base de Datos:** MySQL 8.0.
* **Frontend:** Thymeleaf, HTML5, CSS3 (Diseño moderno/oscuro), JavaScript.
* **QR:** Google ZXing para la generación de códigos.
